## Local Setup
 
To setup and run this project locally follow the steps below.

1. Clone the repo
    ```sh
   git clone https://github.com/sauravUppoor/vaccine-tracker.git
   ```
2. Run 
    ```sh
    npm install
    ```
3. Build
    ```sh
    npm run build
    ```
 4. Run
    ```sh
    serve -s build
    ```
 5. Open the deployed link in browser.
